# Projeto-An-lise-de-Dados-Brazilian-E-Commerce-OLIST
Pipeline completo de Análise de Dados (Excel ➔ Python ➔ Power BI) desenvolvido para mapear o comportamento de vendas, churn e eficiência logística de uma operação de e-commerce com mais de 100 mil registros. O projeto transforma dados brutos em um dashboard gerencial focado na redução de gargalos operacionais e previsibilidade de prazos.
